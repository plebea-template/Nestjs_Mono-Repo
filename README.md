# Description

Template of Monolithic Repository by Yarn Workspaces with Nest.js

# Bootstrap

## Dependency Installation

```
npm i -g yarn
yarn install
```

## Build

```
yarn workspaces foreach build
```

# Start

## Production mode

```
yarn workspace server start:prod
```

## Development mode

```
yarn workspace server start:dev
```
